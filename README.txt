SSRating Extension 1.1.01

 * http://plugins.jquery.com/project/Star_Rating_widget
***********************************************

Requirements
-----------------------------------------------
SilverStripe 2.3.x and 2.4.x

Introduction
-----------------------------------------------
Implements a Star Ranking system. 
Display a Rated and Ranking with stars.
The Ranking is calculated by the user's recorded option 

Installation
-----------------------------------------------
Download and install the project to the root directory of your silverstripe project

To use the SSRanking in a template:

    * $getSSRating
